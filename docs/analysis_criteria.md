# 감정 라벨 및 갈등 위험도 기준표

## 1. 문서 목적

본 문서는 **연인 갈등 상황 대응 감정 기반 RAG 챗봇** 프로젝트에서 사용하는 **감정 분류 기준**과 **갈등 위험도 판단 기준**을 정의한다.

`emotion_analyzer.py`와 `risk_analyzer.py`의 분류 로직은 이 문서에 정의된 기준을 따른다.

## 2. 감정 라벨 기준 정의

### 2-1. 원본 감정 라벨 (7종)

데이터셋(`continuous_dialogue_utterance_cleaned.csv`)에 포함된 원본 감정 라벨은 다음 7종이다.

| 라벨 | 영문 | 설명 | 데이터 분포 |
|---|---|---|---|
| 중립 | neutral | 특별한 감정 표현 없이 사실 전달이나 일상 대화 | 43,789건 (78.7%) |
| 놀람 | surprise | 예상치 못한 사실에 대한 놀라움 반응 | 4,866건 (8.8%) |
| 분노 | anger | 화남, 짜증, 불만, 공격적 표현 | 3,633건 (6.5%) |
| 슬픔 | sadness | 서운함, 우울, 아쉬움, 미안함 | 1,972건 (3.5%) |
| 행복 | happiness | 기쁨, 만족, 감사, 긍정적 반응 | 1,030건 (1.9%) |
| 혐오 | disgust | 짜증, 경멸, 거부감, 냉소적 반응 | 220건 (0.4%) |
| 공포 | fear | 불안, 걱정, 두려움, 염려 | 98건 (0.2%) |

### 2-2. 감정 상위 그룹 (3그룹)

챗봇 응답 전략 수립을 위해 7종 라벨을 **3개 상위 그룹**으로 재분류한다.

| 상위 그룹 | 포함 라벨 | 설명 | 응답 전략 |
|---|---|---|---|
| **부정(negative)** | 분노, 슬픔, 혐오, 공포 | 갈등 악화 가능성이 있는 감정 | 공감 표현 + 진정 유도 |
| **중립(neutral)** | 중립 | 감정 표현이 약하거나 없는 상태 | 상황 파악 + 부드러운 질문 |
| **긍정(positive)** | 행복, 놀람 | 갈등 완화 또는 긍정적 전환 가능 | 긍정 강화 + 해결 유도 |

> **참고**: `놀람(surprise)`은 맥락에 따라 긍정·부정 모두 가능하나, 데이터 분석 결과 대부분 중립~긍정적 놀라움이므로 **긍정** 그룹으로 분류한다. 향후 문맥 기반 세분화 가능.

### 2-3. 감정 강도 산출 기준

발화 단위의 감정 외에, **대화 전체의 감정 강도**를 수치화하여 위험도 판단의 입력으로 사용한다.

| 지표 | 산출 방식 | 설명 |
|---|---|---|
| `negative_ratio` | (부정 감정 발화 수) / (전체 발화 수) | 대화 내 부정 감정 비율 |
| `anger_ratio` | (분노 발화 수) / (전체 발화 수) | 분노 감정 집중도 |
| `emotion_volatility` | 감정 전환 횟수 / (전체 발화 수 - 1) | 감정 변동성 (잦은 전환 = 불안정) |
| `negative_streak` | 연속 부정 감정 최대 길이 | 부정 감정 연속 정도 |
| `ending_sentiment` | 마지막 발화의 감정 그룹 | 대화 종료 시 감정 방향 |

## 3. 갈등 위험도 기준 정의

### 3-1. 위험도 등급 (5단계)

대화의 갈등 위험도를 **5단계**로 정의한다.

| 등급 | 라벨 | 점수 범위 | 설명 | 챗봇 대응 방향 |
|---|---|---|---|---|
| 1 | **안전(safe)** | 0.0 ~ 0.2 | 갈등 징후 없음. 일상 대화 수준 | 일반 응답 |
| 2 | **주의(caution)** | 0.2 ~ 0.4 | 경미한 불만/서운함 표현 | 공감 표현 + 상황 확인 |
| 3 | **경고(warning)** | 0.4 ~ 0.6 | 명확한 갈등 신호. 분노/혐오 표현 반복 | 적극적 공감 + 진정 유도 |
| 4 | **위험(danger)** | 0.6 ~ 0.8 | 강한 갈등. 공격적 표현, 감정 폭발 | 긴급 공감 + 대화 전환 시도 |
| 5 | **심각(critical)** | 0.8 ~ 1.0 | 극단적 갈등. 관계 단절 위험 표현 | 전문 상담 권유 + 안전 대응 |

### 3-2. 위험도 점수 산출 방식

위험도 점수(0.0 ~ 1.0)는 다음 **5개 시그널**의 가중 합산으로 산출된다.

| 시그널 | 가중치 | 산출 방식 |
|---|---|---|
| 부정 감정 비율 (`negative_ratio`) | 0.25 | 부정 감정 발화 비율 (0~1) |
| 분노 집중도 (`anger_ratio`) | 0.20 | 분노+혐오 발화 비율 (0~1) |
| 위험 키워드 탐지 (`keyword_score`) | 0.25 | 위험 키워드 매칭 탐지 점수 (0~1) |
| 부정 감정 연속성 (`streak_score`) | 0.15 | 연속 부정 감정 길이 기반 점수 (0~1) |
| 대화 종료 감정 (`ending_score`) | 0.15 | 마지막 발화 감정 기반 점수 (0~1) |

**최종 점수** = Σ(시그널 × 가중치), 클리핑 [0.0, 1.0]

### 3-3. 위험 키워드 사전

갈등 위험도 판단의 핵심 보조 지표인 **위험 키워드**를 다음과 같이 분류한다.

#### 등급 1: 극위험 키워드 (가중치 1.0) — `critical`

폭력·자해·자살 암시, 극단적 행위 위협. **즉각적인 안전 개입이 필요한 표현.**

| 하위 분류 | 키워드 |
|---|---|
| 자해/자살 암시 | 죽겠다, 죽고싶, 죽을래, 죽어버릴, 살기싫, 못살겠, 목숨, 유서 |
| 폭력 위협 | 때리, 때릴, 패버릴, 죽여버릴, 없애버릴, 찔러, 칼, 폭력 |
| 극단적 행동 예고 | 뛰어내릴, 끝장, 다죽자, 같이죽자 |

#### 등급 2: 고위험 키워드 (가중치 0.8) — `high`

관계 단절 선언, 강한 거부·추방 표현. **관계의 존속 자체를 위협하는 표현.**

| 하위 분류 | 키워드 |
|---|---|
| 관계 단절 선언 | 헤어지자, 이별, 끝내자, 끝이야, 끝났어, 그만하자, 이혼, 갈라서자, 나가살아, 각자살자 |
| 강한 거부·추방 | 꺼져, 사라져, 보기싫어, 지긋지긋, 질렸어, 상종, 인연끊, 연락하지마, 다신보지말자 |
| 결정적 포기 | 포기, 체념, 미련없 |

#### 등급 3: 중위험 키워드 (가중치 0.6) — `medium`

인격 비난, 경멸, 모욕성 발언. **상대방 자존감을 직접 공격하는 표현.**

| 하위 분류 | 키워드 |
|---|---|
| 인격 비하·모욕 | 한심, 찌질, 못났, 쓸모없, 무능, 멍청, 바보같, 저질, 수준이하, 한심하기 |
| 경멸·혐오 | 싸가지, 재수없, 꼴불견, 역겹, 구역질, 못생, 징그럽, 더럽 |
| 비열·도덕적 비난 | 비열, 치사, 뻔뻔, 파렴치, 양심없, 이기적, 매너없, 인간이하, 쓰레기 |
| 신뢰 파괴 | 배신, 배신자 |

#### 등급 4: 주의 키워드 (가중치 0.4) — `elevated`

강한 불만·짜증, 불신 표현. **갈등이 명확하나 관계 단절까지는 아닌 표현.**

| 하위 분류 | 키워드 |
|---|---|
| 강한 감정 분출 | 짜증, 열받, 미치겠, 환장, 빡치, 빡쳐, 화나, 화났, 폭발, 참을수없 |
| 불신·의심 | 거짓말, 뻥, 속이, 거짓, 믿을수없, 의심, 수상, 숨기, 감추 |
| 공격적 명령·비난 | 닥쳐, 입다물, 나가, 말걸지마, 구박, 못참, 안돼, 안져 |
| 극단적 부정 표출 | 못견디겠, 견딜수없, 미칠것같, 돌아버리겠 |

#### 등급 5: 저위험 키워드 (가중치 0.2) — `low`

경미한 서운함, 실망, 소극적 불만. **갈등 초기 신호이나 직접적 공격성은 없는 표현.**

| 하위 분류 | 키워드 |
|---|---|
| 서운함·실망 | 서운, 섭섭, 실망, 아쉽 |
| 답답함·속상함 | 답답, 속상, 걱정, 불안, 우울, 힘들, 지치 |
| 소극적 불만 | 왜그래, 왜이래, 그러지마, 하지마, 그만해, 됐어, 알았어, 몰라 |
| 회피·냉담 | 말하기싫, 귀찮, 상관없, 알아서해 |

## 4. Rule-Based 분류 vs LLM 분류

본 프로젝트는 **이중 분류 체계**를 채택한다.

### 4-1. Rule-Based 분류 (기본)

- 키워드 매칭 + 감정 시퀀스 통계를 활용한 규칙 기반 분류
- **장점**: 빠른 속도, 결정론적 결과, 해석 가능성
- **단점**: 문맥 파악 한계, 비유/은유/반어법 대응 불가
- **적용**: 실시간 1차 필터링, 위험도 스크리닝

### 4-2. LLM 기반 분류 (고급)

- LLM의 **언어 추론·논리 능력**을 활용한 정밀 분류
- **장점**: 문맥 이해, 비유·반어 파악, 복합 감정 탐지
- **단점**: 속도, 비용, 비결정론적 결과
- **적용**: Rule-Based 결과의 검증, 경계선 사례 재분류

### 4-3. 검증 전략

LLM 분류 결과의 신뢰성 검증을 위해 다음을 적용한다.

1. **Chain-of-Thought (CoT) 추론**: LLM이 분류 근거를 단계별로 명시
2. **구조화 출력 강제**: JSON 형식으로 결과 반환을 강제
3. **교차 검증**: Rule-Based 결과와 LLM 결과 불일치 시 재검토 트리거
4. **신뢰도 점수**: LLM이 자체 신뢰도(confidence)를 함께 반환

## 5. 출력 구조 정의

### 5-1. 감정 분석 결과 구조

```json
{
  "utterance": "원본 발화 텍스트",
  "emotion": {
    "primary": "분노",
    "primary_en": "anger",
    "group": "negative",
    "confidence": 0.85
  },
  "method": "rule_based | llm",
  "reasoning": "분류 근거 설명 (LLM 사용 시)"
}
```

### 5-2. 위험도 분석 결과 구조

```json
{
  "dialogue_id": "대화 ID",
  "risk_score": 0.72,
  "risk_level": "danger",
  "risk_label": "위험",
  "signals": {
    "negative_ratio": 0.65,
    "anger_ratio": 0.40,
    "keyword_score": 0.80,
    "streak_score": 0.60,
    "ending_score": 1.0
  },
  "detected_keywords": [
    {"keyword": "꺼져", "severity": "high", "weight": 1.0}
  ],
  "emotion_sequence": ["분노", "중립", "분노", "혐오", "분노"],
  "recommendation": "긴급 공감 + 대화 전환 시도"
}
```
